2. 由BA牵头，改完一个文档自动同步到自己组的开发人员的指定目录。
   BA一改就能同步，同步时要不要经过开发的确认？
   

3. 传输大文件的功能，解决邮件和快乐平安不同传输大文件的问题


4. 区块链，文件校验机制
